# Themes

## COLORS

1. You can access colors from `from axto.styles import Color`
2. Available colors
    1. BLACK, RED, GREEN, YELLOW, BLUE, MAGENTA, CYAN, WHITE
    2. BRIGHT_BLACK, BRIGHT_RED, BRIGHT_GREEN, BRIGHT_YELLOW, BRIGHT_BLUE, BRIGHT_MAGENTA, BRIGHT_CYAN, BRIGHT_WHITE
    3. BOLD, DIM, UNDERLINE
3. Available theme set
    1. default_text         <- uses to print text
    2. border_normal        <- uses to draw border of box
    3. border_focus         <- uses to draw border of box when selected
    4. widget_selected      <- uses when widget is selected (except box)
    5. widget_deselected    <- uses when widget is normal (except box)
    6. placeholder          <- uses for drawind placeholders
    7. progress_fill        <- uses when progress bar is not completed
    8. progress_complete    <- uses when progress bar is completed
    9. list_item_selected   <- uses when item of scroll list is selected
    10. list_item_normal    <- uses when item of scroll list is normal
